  1. End point for shortening the url : POST /shorten
    Request body - {“url”:”http://a.very.long.url“}
    2. curl -sX POST -H 'Content-Type: application/json' 'localhost:8080/shorten' -d 
    '{"url":"http://a.very.long.url"}' 
    

{"short":"http://localhost:8080/original/000001",
    
"redirect":"http://localhost:8080/000001"}
  2. End point for getting original url : GET /original/{short_url} 
    curl -sX GET -H 'Content-Type: application/json' 'localhost:8080/original/000001' 
    

{"original":"http://a.very.long.url"}
  3. Redirect to original( long) url : GET /{short_url} 
    curl -sX GET -H 'Content-Type: application/json' 'localhost:8080/000001'
